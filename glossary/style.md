# Translation Style Guide

## Non-Negotiable Rules

- Translate visible text only.
- Preserve Markdown structure, links, images, and file names.
- Do not invent new HP canon names when Potter's Army / ROSMEN already has an
  established form.
- Use `Гарри Поттер и методы рационального мышления` and `ГПиМРМ` as the
  project's title conventions.
- Use `hpmor.ru` chapter-title precedent when a fanfic references, mirrors, or
  parodies HPMOR chapter naming.

## Tone

- Write natural literary Russian.
- Aim for the precision and readability associated with good Russian Harry
  Potter prose, without overriding approved canon naming.
- Preserve humor, pacing, and character voice.
- Avoid flattening distinctive narration into neutral machine prose.

## Formatting

- Keep paragraph boundaries unless the source markup is clearly broken.
- Preserve emphasis, scene breaks, lists, and quotes.
- Do not infer extra headings that are not already present in the imported
  markdown.

## Workflow

- Check `glossary/glossary.md` before translating a segment.
- If a recurring term is missing, add it to the glossary before continuing.
- Validate often and build preview EPUBs during translation.
- Treat glossary mismatches as a review blocker even when the output reads well.
