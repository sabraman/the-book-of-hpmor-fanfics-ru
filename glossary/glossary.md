# Glossary

This file is the canonical terminology authority for the project.
Every recurring term used in translation should appear here before it is used
across multiple segments.

## Authority Matrix

1. Potter's Army for Harry Potter canon names, places, spells, institutions,
   and franchise terminology.
2. ROSMEN alignment through Potter's Army canon choices.
3. `hpmor.ru` for the HPMOR Russian title, chapter-title conventions, and
   established HPMOR-specific wording.
4. Spivak for sentence-level accuracy and literary naturalness after canon
   naming is already fixed.

## Usage Rules

- Do not invent new Russian variants for canon entities when a Potter's Army /
  ROSMEN form exists.
- If a fanfic-specific term has no established precedent, add it here before
  reusing it in multiple files.
- Chapter titles that intentionally echo HPMOR should follow the established
  wording pattern from `hpmor.ru` where applicable.
- For wizarding idioms and exclamations, prefer established Russian idiomatic
  forms over literal calques. For example, `Dear Merlin` may become `Мерлинова
  Борода` in exclamatory dialogue, while context-specific forms like `ради
  Мерлина`, `во имя Мерлина`, `слава Мерлину`, and `упаси Мерлин` should be
  chosen by sense.
